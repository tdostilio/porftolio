Featured Project

New Story

New Story is a San Francisco based, Y-Combinator funded, non-profit that builds homes for families in impoverished nations. A teammate and I helped them build out their donation platform by creating the ability to donate as a team. A team leader signs up, and invites his or her friends to collectively donate monthly towards the construction of a home. It was a unique experience to engineer a platform for a live client while still in school, and it was for an incredible cause. Please click the Demo button below to see the "Build a Home With Friends" feature, or learn more about the charity by clicking the button to visit New Story's site.

Technologies: jQuery, Node.js, Express, Handlebars, Stripe API, Nodemailer, PostgreSQL, Heroku

Projects

Glacion

Glacion is an application that helps people find the locations of specific job concentrations in an area. This helps users decide where to live if they are looking for a new job in a city other than where they live. The user is prompted to input a city they would like to live in, a job they would like to have, and a time frame the job has been listed in. Glacion takes the user's information and displays a heat map overlay on a Google map. High heat areas show where jobs are concentrated. This gives the user an idea of where they should move to.

Technologies: JavaScript, jQuery, GoogleMaps API, HTML, CSS, Dice API

Championship Race Challenge

Written in Python using the Pygame library this game is a simple 2d racing game. There is a starting point and an ending point, in this case represented by a trophy. Contact with the trophy in the allotted time will satisfy the win condition and allow you to proceed. Collision with any of the wall barriers or an expiration of the timer will fail you. It was a great experience writing in another language and a very fun final product.

Technologies: Python, Pygame

The Movie Wizard

The Movie Wizard is a web application that returns movies and their relevant information based off a title search. It presents the user with information including box-office sales, Rotten Tomatoes score, and main actors. There is a "Suggestions" button that presents the user with similar movie titles, which is helpful if you can't decide what to watch! This is a project I hope to continually build on and add features to, such as video game support and a suggestion AI.

Technologies: JavaScript, jQuery, HTML, CSS, OMDB API, TMDB API
